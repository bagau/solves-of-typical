Для создания сайта с фиксированной шириной нужно создать примерно такую структуру
Нужны такие классы .l-container { min-width: 1355px; } .l-wrapper { width: 1300px; margin: 0 auto; } При этом горизонтальная прокрутка появится при 1355 пикселей. Блок l-wrapper будет оцентрирован, по краям останутся отступы (1355 - 1300)/2 = 27.6 пикселей. Не будет обрезания фонов при появлении горизонтальной прокрутки
